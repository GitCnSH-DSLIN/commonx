Declaration

type TIntGrayAlphaRec = record

Description

Alternative representation of TIntGrayAlpha, where each element can be accessed as an individual value. This can be safely typecast to TIntGrayAlpha and vice-versa.

Overview

Fields

Gray: Byte;
Alpha: Byte;
Values: packed[0..1] of Byte;

Description

Fields

Gray: Byte;

Gray value ranging from 0 (black) to 255 (white).

Alpha: Byte;

Alpha-channel value ranging from 0 (translucent) to 255 (opaque).

Values: packed[0..1] of Byte;

Values represented as an array, where gray has index of 0 and alpha-channel index of 1.
