type TFloatGrayAlpha = record

Description

Grayscale floating-point color representation, where each of the components range from 0 to 1. Values outside of this range are also supported, but they cannot be reliably displayed on the screen.

Overview

Fields

Gray: VectorFloat;
Alpha: VectorFloat;
Values: array[0..1] of VectorFloat;

Description

Fields

Gray: VectorFloat;

Gray value ranging from 0.0 (black) to 1.0 (white).

Alpha: VectorFloat;

Alpha-channel value ranging from 0.0 (translucent) to 1.0 (opaque).

Values: array[0..1] of VectorFloat;

Values represented as an array, where gray has index of 0 and alpha-channel index of 1.
